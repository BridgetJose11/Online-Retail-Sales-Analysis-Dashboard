# Online-Retail-Sales-Analysis-Dashboard
 

Overview

This project analyzes an online retail dataset using Excel and Power BI. It focuses on data cleaning, transformation, and building an interactive dashboard to generate meaningful business insights. The dataset is sourced from the UCI Machine Learning Repository. The dataset contains transactional records of a UK-based online retail business from December 2010 to December 2011. The company sells unique gift items, and many customers are wholesalers, leading to high-volume and repeat purchases.

Objectives

Perform data preprocessing (cleaning & transformation), analyze sales trends and customer behavior, build an interactive dashboard, and communicate insights using visualizations

Tools Used: Microsoft Excel and Power BI

Data Preprocessing Steps

Removed duplicates, handled missing values, filtered invalid data (negative quantity, zero price), standardized formats, and extracted Month and Year

Dashboard Features

Cards (Total Sales, Orders, Customers), Sales by Product (Bar Chart), Monthly Sales Trend (Line Chart), Sales Distribution by Country (Pie Chart), Top Customers Analysis, Interactive slicers (Country, Month, Year)

Key Insights

Sales peak during November–December (holiday season), the United Kingdom generates the highest revenue, top customers contribute significantly to total sales, and strong seasonal buying patterns are observed

Conclusion

This project demonstrates how data analysis and visualization can provide actionable insights for business decision-making.
